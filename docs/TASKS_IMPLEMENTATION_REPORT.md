## 🤖 Интеграция с AI (следующий этап)

### Function Calling

AI сможет создавать задачи через Mistral Function Calling:

```python
TOOLS = [
    {
        "type": "function",
        "function": {
            "name": "create_task",
            "description": "Создаёт новую задачу",
            "parameters": {
                "type": "object",
                "properties": {
                    "title": {"type": "string"},
                    "description": {"type": "string"},
                    "due_date": {"type": "string", "format": "date-time"},
                    "priority": {"type": "string", "enum": ["low", "medium", "high", "urgent"]}
                },
                "required": ["title"]
            }
        }
    }
]
```

### Примеры использования

**Пользователь:**
```
Карина, создай задачу "Подготовить отчёт" на пятницу
```

**AI:**
```json
{
  "tool_calls": [
    {
      "function": {
        "name": "create_task",
        "arguments": {
          "title": "Подготовить отчёт",
          "due_date": "2026-03-13T23:59:59Z",
          "priority": "high"
        }
      }
    }
  ]
}
```

**Результат:**
```
✅ Задача создана!

🔴 🟠 Подготовить отчёт
📅 Дедлайн: 13.03.2026 (пятница)
⏱ Оценка: 4ч
```
