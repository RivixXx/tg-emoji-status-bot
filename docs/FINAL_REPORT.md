## 🎯 Обзор проекта

**Цель:** Превратить Karina AI из реактивного бота в **проактивного AI-ассистента** для повышения продуктивности.

**Результат:** Полноценная система управления задачами, проектами, спринтами с AI-интеграцией и проактивными уведомлениями.

### 7 типов триггеров:

| Время | Триггер | Описание |
|-------|---------|----------|
| **7:00** | 🌅 Утреннее планирование | План на день, встречи, дедлайны, спринт |
| **9:00** | 🔴 Просроченные задачи | Проверка должников |
| **12:00** | 🔍 Застрявшие задачи | Задачи 3+ дня в todo |
| **13:00** | 🍽️ Обед | Напоминание поесть |
| **:30** | ☕ Перерывы | Каждый час (только будни) |
| **:00** | ⏰ Дедлайны | Проверка каждый час (за 24ч, за 1ч, сейчас) |
| **20:00** | 🌙 Вечерний обзор | Выполненные задачи, итоги дня |

### Особенности:

- ✅ **Защита от дублирования** — state-переменные
- ✅ **Проверка времени** — ±1 минута
- ✅ **Только будни** — для перерывов и обеда
- ✅ **Inline кнопки** — быстрые действия
- ✅ **Логирование** — каждое срабатывание
- ✅ **Graceful degradation** — при ошибках

### Примеры уведомлений:

**Утро (7:00):**
```
☀️ Доброе утро, Михаил!

📍 Встречи:
  • 10:00 — Команда
  • 14:00 — Клиент

⚠️ Дедлайны на сегодня:
  🔴 Подготовить отчёт

🏃 Активный спринт: Март 2026
  ⏳ Осталось дней: 5
  📊 Прогресс: 60%
```

**Перерыв (14:30):**
```
☕ Время перерыва!

5 минут отдыха повысят продуктивность. Разомнись!

[✅ Уже иду!] [⏰ Позже (15 мин)]
```

### Новые AI инструменты:

1. **`create_task`** — создание задачи голосом
2. **`get_my_tasks`** — получение списка задач
3. **`complete_task`** — завершение задачи
4. **`create_project`** — создание проекта
5. **`get_sprint_info`** — информация о спринте
6. **`get_productivity_stats`** — статистика продуктивности

### Примеры использования:

**Пользователь:**
```
Карина, создай задачу "Подготовить отчёт" на пятницу с высоким приоритетом
```

**AI (Function Calling):**
```json
{
  "tool_calls": [{
    "function": {
      "name": "create_task",
      "arguments": {
        "title": "Подготовить отчёт",
        "description": "Квартальный отчёт по продажам",
        "due_date": "2026-03-13T23:59:59Z",
        "priority": "high"
      }
    }
  }]
}
```

**AI (ответ):**
```
✅ Задача создана!

📝 Подготовить отчёт
📅 Дедлайн: 13.03.2026 23:59
🔴 Приоритет: high
```

## 🔧 Технические детали

### База данных:
- **PostgreSQL** с Supabase
- **7 таблиц** (projects, tasks, sprints, sprint_tasks, daily_goals, task_comments, task_history)
- **20+ индексов для оптимизации
- **RPC функции** для эффективных запросов
- **Триггеры** для авто-обновления и логирования

### AI:
- **Mistral AI** (mistral-small-latest)
- **Function Calling** с 17 инструментами
- **RAG память** (векторный поиск)
- **Контекст диалога** (10 сообщений)
- **Circuit Breaker** для отказоустойчивости

### Триггеры:
- **Асинхронный цикл** (проверка каждую минуту)
- **State-переменные** для защиты от дублирования
- **Graceful degradation** при ошибках
- **Логирование** каждого срабатывания
